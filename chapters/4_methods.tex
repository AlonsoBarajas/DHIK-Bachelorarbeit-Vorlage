\section{Enumerations and items}

\LaTeX{} also has different environments to create list with numbers or bulletpoints. Here are some examples.
\\\textbf{Bulletpoints}
\begin{itemize}
    \item Item 1
    \item Item 2
    \item Item 3
    \begin{itemize}
        \item Item A
        \item Item B
        \item Item C
    \end{itemize}
    \item Item 4
    \item Item 5
\end{itemize}

\textbf{Enumerations}
\begin{enumerate}
    \item Item 1
    \item Item 2
    \item Item 3
    \begin{enumerate}
        \item Item A
        \item Item B
        \item Item C
    \end{enumerate}
    \item Item 4
    \item Item 5
\end{enumerate}
